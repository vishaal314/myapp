## SAMENVATTING (248 WOORDEN)

Een geautomatiseerd Data Protection Impact Assessment (DPIA) systeem voor GDPR Article 35 compliance verificatie. De uitvinding implementeert 5-categorie risk assessment: data category (sensitive data, children, vulnerable persons), processing activity (automated decisions, systematic monitoring), rights impact (discrimination, harm), transfer/sharing (international transfers), security measures (encryption, access controls). Totaal 25 assessment questions verdeeld over categorieën.

Automated risk scoring algorithm met thresholds: high≥7 (DPIA mandatory), medium≥4 (DPIA recommended), low<4 (DPIA optional). DPIA requirement determination via formule: dpia_required = (overall_risk == "High") OR (high_risk_count >= 2) OR (file_high_risk > 0).

**Unique feature**: Code repository integration scant GitHub repositories, local file systems, uploaded files voor automated PII detection. Technical + legal assessment combined waarbij code findings questionnaire scores overriden. PII patterns: email, phone, credit card, BSN (Netherlands 9-digit + checksum), IP addresses.

Enhanced real-time monitoring integreert comprehensive GDPR validator (Articles 25, 30, 35, 37, 44-49), EU AI Act compliance checker, Netherlands UAVG validator. Automatic risk escalation bij critical violations.

Netherlands specialization: Dutch language support (Nederlands/Engels), AP (Autoriteit Persoonsgegevens) authority integration met notification templates en verification URLs, BSN detection met 11-proef checksum validatie, regional penalty multipliers.

Automated recommendation engine genereert category-specific guidance met technical remediation: BSN removal, hardcoded credentials fix, encryption upgrades. Article-specific references (GDPR Art. 9, 25, 32, 35).

Performance: 2 hours versus 20 hours manual (90% time savings), €500-€2,000 versus €5,000-€25,000 cost reduction. Competitor gap: OneTrust/TrustArc lack code repository scanning.
