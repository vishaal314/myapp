SSH Key Generation:
==================

On your local machine, run:

# Generate SSH key pair
ssh-keygen -t rsa -b 4096 -f ~/.ssh/dataguardian_deploy

# Copy public key to your server
ssh-copy-id -i ~/.ssh/dataguardian_deploy.pub root@45.81.35.202

# Copy private key content for GitHub secret
cat ~/.ssh/dataguardian_deploy

Then paste the ENTIRE private key content (including -----BEGIN and -----END lines) 
into the SERVER_SSH_KEY secret in GitHub.
