SECTION 2: INVENTION DETAILS
================================================================================

Title of the invention:
INTELLIGENT DATABASE SCANNER WITH ADAPTIVE SAMPLING AND PRIORITIZATION FOR 
PII DETECTION

Technical field:
Information Technology; Data Protection; Privacy Compliance; 
Database Security; GDPR Compliance Tools

Brief description (max 150 words):
An intelligent database scanner that automatically detects Personally 
Identifiable Information (PII) in relational databases with three adaptive 
scanning modes: FAST (100 rows, 2 workers), SMART (300 rows, 3 workers), 
and DEEP (500 rows, 3 workers). The system uses an advanced prioritization 
system that identifies high-risk tables (user/customer/employee: 3.0×, 
medical: 3.0×, payment: 2.8×) and applies parallel scanning for optimal 
performance. Supports PostgreSQL, MySQL, and SQL Server with Netherlands-
specific PII detection including BSN validation according to the official 
11-proef algorithm. Delivers detailed GDPR compliance reports with article 
references and risk analysis.
