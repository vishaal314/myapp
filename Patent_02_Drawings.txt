FIGURE 3: BSN 11-PROEF VALIDATION ALGORITHM
================================================================================

[See Patent_02_Tekeningen.txt for complete algorithm visualization]

Step-by-step process:
1. Input validation (9 digits, numeric only)
2. Position indexing (0-8)
3. Checksum calculation with multipliers:
   - Positions 0-7: multiplier = (9-i)
   - Position 8: multiplier = -1
4. Modulo 11 validation
5. Result: VALID (mod 11 = 0) or INVALID (mod 11 ≠ 0)

Example calculation for BSN 111222333:
(1×9) + (1×8) + (1×7) + (2×6) + (2×5) + (2×4) + (3×3) + (3×2) - (3×1) = 66
66 mod 11 = 0 → VALID

Python implementation included showing:
- validate_bsn_11_proef() function
- Input validation logic
- Checksum calculation loop
- Modulo 11 verification
- Test cases (valid/invalid BSN numbers)

Integration with database scanner:
- Regex pattern detection: r'\b\d{9}\b'
- Validation call for each match
- False positive prevention
- GDPR Article 9 mapping
