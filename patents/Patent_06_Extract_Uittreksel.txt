Cloud Sustainability & Carbon Footprint Scanner

===============================================================================

TITEL
Cloud Sustainability & Carbon Footprint Scanner with Zombie Resource Detection,
Regional CO₂ Emission Calculation, Code Bloat Analysis, and Netherlands Carbon
Tax Compliance

===============================================================================

SAMENVATTING (248 WOORDEN)

Een geautomatiseerd cloud sustainability scanning systeem dat zombie resources 
detecteert, regionale CO₂ emissies berekent, en code bloat analyseert met 
Netherlands carbon tax compliance. Het systeem biedt als EERSTE cloud scanner
geïntegreerde sustainability metrics met privacy compliance voor CSRD 
(Corporate Sustainability Reporting Directive) rapportage.

Zombie resource detection identificeert idle virtual machines (<10% gemiddelde
CPU over 30 dagen), unused storage accounts (0 transactions in 90 dagen), 
orphaned IP addresses (niet gekoppeld), en unattached disks (>30 dagen oud).
Potentiële besparingen: 95% van resource kosten via automated shutdown
recommendations.

Regional CO₂ emission calculation implementeert specifieke emission factors
per regio: EU gemiddelde 0.295 kg CO₂/kWh, Netherlands 0.412 kg CO₂/kWh 
(coal-heavy grid), Norway 0.008 kg CO₂/kWh (hydropower), Germany 0.338 kg
CO₂/kWh, France 0.056 kg CO₂/kWh (nuclear). VM power consumption berekent
via SKU-specific wattage (Standard_B1s 25W, Standard_D8s_v3 320W) × CPU
utilization percentage.

Code bloat analysis via AST parsing detecteert dead functions (defined maar
never called), duplicate code blocks (>80% similarity over 10+ lines), unused
imports (imported maar never used), en large binary files (>10 MB). Bloat
score calculation: dead_functions × 10 + duplicate_blocks × 15.

Multi-cloud support voor Azure (ARM API), AWS (CloudWatch), en GCP (Cloud
Monitoring). Netherlands CO₂-heffing industrie compliance met rates €30-80/
tonne CO₂ (2024-2030). Carbon cost calculation: (total_co2_kg / 1000) × €60
carbon tax = financial impact. CSRD Scope 3 emissions categorization voor
cloud provider carbon footprint.

Unieke market moat: EERSTE compliance tool met integrated sustainability +
carbon tracking. Typische organisatie bespaart €4,580/maand + 380 kg CO₂
reduction (equivalent 17.5 bomen).
